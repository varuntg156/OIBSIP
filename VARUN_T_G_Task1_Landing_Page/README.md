# ☕ Coffee Website Landing Page

A modern and responsive coffee shop landing page developed using HTML, CSS, and JavaScript. This project was created as part of the Oasis Infobyte Web Development Internship.

## 📌 Project Overview

The Coffee Website Landing Page is designed to provide an attractive and engaging user experience for a coffee brand. The website features a clean layout, responsive design, and visually appealing sections to showcase products and services.

## 🚀 Features

- Responsive landing page design
- Modern user interface
- Navigation bar
- Hero section
- Product and service sections
- Footer section
- Mobile-friendly layout
- Smooth and organized styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
